# HTML Complete Practice Repository

This repository contains a complete step-by-step guide to learning HTML from scratch.

## What you will learn
- HTML basic structure
- Text elements
- Links & images
- Lists
- Tables
- Forms
- Semantic HTML

## How to use
1. Start from `index.html`
2. Follow folders in numeric order
3. Open files in browser and read comments
